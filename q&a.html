Question-1: What are the different ways to select an element in the DOM?
Elements can be selected using getElementById for IDs, getElementsByClassName for classes, getElementsByTagName for tags, querySelector for the first match of a CSS selector, and querySelectorAll for all matches. You can also traverse from parent elements using parent.querySelector().
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML returns HTML content including tags, innerText returns only visible text respecting CSS, and textContent returns all text ignoring HTML and CSS. textContent is faster, innerHTML can include HTML, and innerText considers what is visible.
Question-3: What is event delegation in the DOM?
Event delegation is attaching one event listener to a parent element to handle events on its child elements. It improves efficiency and works for dynamically added elements by checking the event target when the event bubbles up.
Question-4: What is event bubbling in the DOM?
Event bubbling is when an event starts from the target element and propagates up through parent elements to the root. It is default behavior and can be stopped using event.stopPropagation().
Question-5: How do you create, add, and remove elements using JavaScript?
Elements are created using document.createElement(), added using appendChild() or insertBefore(), and removed using remove() or removeChild(). This allows dynamic modification of page content.
